
# 🚀 Exoplanet Exploration Dashboard
An interactive Power BI dashboard designed to analyze and visualize the NASA Exoplanet Archive, covering over 6,000 discovered worlds.

## 🖼️ Dashboard Preview

## 📊 Key Insights & Features
- **Dynamic Analysis:** Interactive filters for Discovery Year and Planet Type.
- **Scientific Visualization:** Used Logarithmic scales to represent astronomical distances and temperatures.
- **Custom Tooltips:** Enhanced user experience with hover-over data summaries for each planet.
- **Technique Analysis:** Visualized the efficiency of various planetary detection methods.

## 🛠️ Tools & Technologies
- **Power BI Desktop:** Dashboard design and visualization.
- **Power Query:** Data cleaning and transformation.
- **NASA Exoplanet Archive:** Source dataset.
